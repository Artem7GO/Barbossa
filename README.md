### Inject:
```
git clone https://github.com/sonicrules11/Barbossa 
cd Barbossa
Open Xcode build 
Build it 
Open Products folder
git clone https://github.com/scen/osxinj
Build it and drag to Barbossa folder
Tab back to terminal
sudo ./osxinj csgo_osx64 libvHook.dylib
```
